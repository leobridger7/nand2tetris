# nand2tetris

A repo containing nand2tetris course untertakings.

Resources:

The nand2tetris course has a few resources. 

- There's the book itself, which is split up into 13 chapters. Each chapter has a project to go with it.

- There's an online IDE, with various components to run the project work, at https://nand2tetris.github.io/web-ide/chip. This can be pointed to the local projects folder in the git checkout.

- There's a few demos on a youtube playlist showing how to use some aspects of the IDE / help with bits of project work - https://www.youtube.com/watch?v=FW6Z_Xp2v-c&list=PLYM3zllSC3SVdjWQUfedxssewHRS7EHuA

- There's an appendix in the book contains some help for completing the project work.


Objective:

- Complete the project work for each chapter.

- For each chapter make some summary notes on paper, with diagrams of any HW components specified in the project. (This should be done after project work has been completed as will have better understanding then).


Repo:

The actual project work itself is within the respective folders inside the 'projects' folder. Some useful docs are within the 'docs' folder.
